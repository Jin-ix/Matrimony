# Indian Catholic Matrimony

A real-time Matchmaking and Digital Twin application built with React, TypeScript, and Node.js.

## Tech Stack
- **Frontend:** React 19, TypeScript, TailwindCSS, Vite
- **Backend:** Node.js, Express, Socket.IO
- **Features:** Real-time chat, matchmaking, profile discovery, kitchen table discussions.

## Running the application
```bash
npm run dev
```
